# FocusTrace

FocusTrace is a focus tracking application that visualizes the gap between what you intend to do and what you actually complete.

Instead of another todo app, FocusTrace helps you understand your work patterns and measure whether you actually followed through on your intentions. 

---

## 🧠 Concept

Most productivity apps focus on listing tasks.

FocusTrace tracks something more meaningful:

> the difference between planned intention and real execution

- What you *intended* to do
- What you *actually completed*
- The gap between intention and execution

This creates awareness around focus, distraction patterns and follow-through.

---

## ✨ Features (Current)

- Create tasks (intentions)
- Delete tasks
- Mark tasks as completed
- Backend + frontend integration (React + Express + TypeScript)

## 🚧 In Progress

## 🚀 Getting started

### Frontend
```bash
# install dependencies
cd client
npm install

# start frontend
npm run dev
```

### Backend
```bash
# install dependencies 
cd server
npm install

# start backend
npm run server
```
> Make sure frontend and backend are running in separate terminals.
